C3D(1)代码使用

1.必须安装以下两个python库

tensorflow和pillow

2、下载UCF101库

3、将视频转换成图片 参数是5fps

./list/convert_video_to_images.sh .../UCF101 5
这个脚本文件里面其实主要是用ffmpeg抽取frame

4、将image转换成list

在命令行输入./convert_images_to_list.sh path/to/video 4

脚本中if (( $(jot -r 1 1 $2) > 1 )); 这个条件是什么意思
thenecho “$imagesFolder” $COUNT >> train.list

$1表示命令行第一个参数

5、训练和测试

  1. python train_c3d_ucf101.py will train C3D model. The trained model will saved in models directory.
  2. python predict_c3d_ucf101.py will test C3D model on a validation data set.
